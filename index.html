Projects

Clinical Diabetes Risk Assessment

This project leverages machine learning to predict diabetes risk using clinical parameters. I built a Random Forest classifier and optimized its threshold to prioritize recall (81%), ensuring minimal false negatives. I integrated SHAP for model interpretability, enabling clinicians to understand the influence of features like glucose and BMI. The results are presented via an interactive Streamlit dashboard with real-time recommendations and visualizations.

Tech: Python, Scikit-learn, SHAP, Streamlit

Happiness Insights 2020

Analyzed the 2020 World Happiness Report to study the pandemic's influence on global well-being. Using data from 153 countries, I developed a predictive model to rank happiness scores and applied SHAP analysis to discover that social support was 28% more influential than GDP. The project features a fully interactive dashboard with dynamic plots and country-level insights to support policy recommendations.

Tech: Python, Pandas, NumPy, Seaborn, SHAP, Scikit-learn, Streamlit, Plotly
